## 💡 Customization Quick Tips

### Change Course Title
Edit `index.html`:
```html
<h1>Your New Course Title</h1>
```

### Change Colors
Edit `style.css` (line with `:root`):
```css
--primary-color: #YOUR_COLOR;
--accent-color: #YOUR_COLOR;
```

### Update Course Fee
Edit `payment.html`:
```html
<div class="fee-amount">₹YOUR_AMOUNT</div>
```

### Replace QR Code
1. Generate your payment QR code
2. Save as PNG: `images/qr.png`
3. Website uses it automatically

### Update Statistics
Edit `index.html` stats section:
```html
<div class="stat-number">YOUR_NUMBER</div>
```
