## 💡 Easy Customization

### Change Course Title
Edit `index.html`:
```html
<h1>Your New Course Title</h1>
```

### Change Colors
Edit `style.css`:
```css
:root {
  --primary-color: #YOUR_COLOR;
  --accent-color: #YOUR_COLOR;
}
```

### Update Statistics
Edit `index.html`:
```html
<div class="stat-number">YOUR_NUMBER</div>
```

### Replace QR Code
1. Generate your payment QR code
2. Save as `images/qr.png`
3. Done! Website uses it automatically

### Add Form Fields
Edit `register.html` and add validation in `script.js`
