## 🚀 Quick Start (5 Minutes)

### To Open the Website:
1. **Option A - Direct:** Double-click `index.html`
2. **Option B - Browser:** Drag `index.html` into your browser
3. **Option C - Server:** Use `python -m http.server 8000` then visit `http://localhost:8000`

### To Navigate:
- **Landing Page:** `index.html` - See course overview
- **Registration:** Click "Register Now" button
- **Payment:** Complete registration to reach payment page

## 🔧 Easy Customization

### Change Course Title
Edit `index.html`:
```html
<h1>Your Course Title</h1>
```

### Change Colors
Edit `style.css`:
```css
:root {
  --primary-color: #YOUR_COLOR;
  --accent-color: #YOUR_COLOR;
}
```

### Update Statistics
Edit `index.html`:
```html
<div class="stat-number">YOUR_NUMBER</div>
```

### Replace QR Code
1. Generate payment QR code
2. Save as `images/qr.png`
3. Done!

### Add More Form Fields
Duplicate form-group pattern in `register.html`
